Game Data Analytics Portfolio Project

Riot Games TFT 시즌3 Match Data 기반 메타 구조 분석 및 밸런스 개선 인사이트 도출

- 분석 목표:

소수 시너지에 플레이가 집중되어 게임 다양성 하락 문제에 대한
플레이 다양성 유도를 위한 보상 시스템 설계

[데이터]

원본 데이터: 399,998건
전처리 후 데이터: 328,532건
유효 매치 수: 49,878개
활성 시너지: 23개

[분석 내용]

1. 메타 구조 분석
- 시너지별 Pick Rate 분석
- Top4 진입률 분석
- Pareto Chart 기반 선택 집중도 분석
- Gini, HHI 지표 활용

2. 조합 패턴 분석
- 시너지 조합 상관관계 분석
- 자주 함께 등장하는 조합 식별
- 순위별 레벨 분포 분석

3. 밸런스 개선 아이디어 도출
- 고인기·저성능 시너지 식별
- 저인기·고성능 시너지 발굴
- 보상 가중치 시뮬레이션 수행

[주요 결과]

- 상위 21% (5종) 시너지가 전체 선택의 45%를 독식
- 티어 구분 없이 승률에 안정적인 고효율 시너지 조합을 선택
- 메카 파일럿·잠입자/반군·우주선 등 자주 선택 시너지 조합 확인
- 승률 상위 유저의 95%가 8-9레벨에 쏠려 있어 전략적 다양성이 외면
- 저인기 고성능 시너지 발굴 및 관련 보상 시스템 제안
- 시스템 적용 후 보상 가중치 시뮬레이션 결과, 메타 집중도(HHI) 2.31% 감소효과

[사용 기술]
- Python
- Pandas
- NumPy
- Plotly
- Matplotlib
- SciPy
